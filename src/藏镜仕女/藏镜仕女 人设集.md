# 原神「愚人众·藏镜仕女」创作人设集

## 一、基础身份与关键身体细节

- **全称**：愚人众·藏镜仕女 / 愚人众·安妮雅（通用代号）
- **所属**：愚人众（至冬国）
- **元素**：水
- **力量核心**：水之邪眼 → 操控水镜
- **定位**：愚人众特殊仪仗部队 / 精锐追踪者与刺客
- **编制**：系编制化单位，并非单一角色（存在众多同名个体）
- **背部**：后背刻印愚人众标志的"圣痕"纹样——战斗时衣服条纹与头冠会从暗变亮发光
- **邪眼位置**：腹部——待机时双手交握恰好遮住，战斗时暴露
- **体型**：胸部丰满，腰肢纤细，臀部浑圆

## 二、外观形象（创作中必须维护）

### 整体
- 身材异常高挑修长的成女体型，丰腴成熟，曲线分明

### 肤色与面容
- **肤色**：苍白（至冬国人典型的雪白肌底）
- **唇色**：蓝色/深蓝色口红——**这是藏镜仕女最具辨识度的妆容特征**，在苍白肌肤上如一抹冷艳的颜色标记；嘴唇本身丰润饱满
- **眼部**：**完全遮蔽**——双眼藏于深色薄纱面幕之后，外界无法窥见其眼神与表情。但考据暗示右眼在驱动邪眼瞬间会闪过一瞬白光
- **发型**：所有头发被头巾/头纱完全包裹，不露出任何发丝——有东正教修女的宗教意味，头发被视为「诱惑之物」而刻意隐藏

### 妆容细节

| 妆容部位 | 特征 | 色情创作意义 |
|----------|------|-------------|
| **唇妆** | 蓝色/深蓝色口红，丰润醒目 | 苍白脸上的唯一颜色——**被吻到花掉、蹭到嘴角、咬到红肿时最刺激** |
| **腮红** | 面颊极淡红晕，若隐若现 | 平时看不出的红晕，只有在**体温升高、呼吸急促时才会真正浮上来** |
| **底妆** | 极致苍白，平滑如玉 | 至冬冰雪肌底——**任何一处泛红（脸颊、耳尖、胸部、大腿内侧）都无所遁形** |
| **眼妆** | 不可见（面幕遮蔽） | 完全未知的区域——**面幕被掀开时，底下的眼睛是哭红了还是瞪大了，全由你写** |
| **颈部** | 黑色choker项圈，紧贴立领下 | 环绕颈部的黑色皮质项圈——**牵住它向后拉，她的天鹅颈就不得不反弓** |
| **头发** | 头巾全包，不见一丝发丝 | 宗教禁欲的极致——**扯掉头巾让头发散落的瞬间，就是"仕女"身份崩坏的开始** |

### 服饰（紫 + 靛蓝主色调）

| 部位 | 特征 |
|------|------|
| **帽子** | 巨大的宽檐帽，帽上附深色透明薄纱面幕，垂落遮蔽全脸——**最标志性特征** |
| **面幕** | 深色薄纱从帽檐垂下，仅露下半张脸的轮廓和唇部 |
| **上衣** | 紧身胸衣，紫靛蓝相间 + 繁复金/银色装饰纹路 |
| **露肩** | 双肩和锁骨完全裸露，大面积苍白肌肤暴露在外 |
| **立领** | 分离式独立立领环绕颈部 |
| **毛皮** | 领口和肩部边缘有白色/灰色毛皮镶边 |
| **袖子** | **极其宽大蓬松的飘袖**，行动时随之飘动 |
| **手部** | 过肘长手套，深色材质覆盖前臂 |
| **下装** | 紧身包臀裙，勾勒腰臀曲线；侧面高开叉，露出大腿 |

### 随身特征
- 身体周围始终悬浮发光的**蓝色水镜碎片和水晶棱柱**，散发幽冷水色光芒
- 操控水镜时可召唤更多折射棱镜环绕

## 三、性格特质

| 关键词 | 对创作的意义 |
|--------|--------------|
| **端庄** | 举手投足带有侍奉神明般的仪式感——**性爱中这种端庄的崩坏本身就是最强的刺激** |
| **冷漠** | 情绪收敛极深，面幕遮眼更添疏离——**撕开这层冰壳时的反差是高潮点** |
| **魅惑** | 水镜之力让猎物心甘情愿坠入幻境——**性挑逗是她的本能武器** |
| **执念深重** | 双眼不知见识过什么，水镜中倒映着不可测的执妄——**有可以被利用的弱点/欲望** |
| **绝对忠诚** | 对女皇的忠诚达到宗教狂热——**可以成为被调教/打破信仰的素材** |
| **纪律严明** | 钢铁般的纪律性——**越是自律的人，失控时的反应越剧烈** |

### 人物气质速写
- **「仿如侍奉神明般的端庄姿态」**——官方定调
- 优雅与危险并存，姿态越从容，手段越致命
- **「似乎一切都逃不过她的耳朵」**——听力极其敏锐，面幕后的感官可能异于常人

## 四、语言风格与台词（创作中必须拿捏的说话方式）

### 语气总览
- 语气**冰冷、从容、不紧不慢**，语速偏慢，带着居高临下的优雅
- 惯用**短促的冷哼**（"哼"、"呵"、"哈"）作为句首发语词——所有情绪中轻蔑是最常被表达的一种
- 声线成熟而有磁性，语调平稳，极少有明显情绪波动——**暴怒时也只是微微加重语气**
- 绝不会说粗俗的话——**她的粗口就是"无礼者"，这是她能说出的最重的话**

### 三种语域（按场景切换）

| 语域 | 触发场景 | 语言特征 | 典型台词 |
|------|----------|----------|----------|
| **宗教祝祷式** | 调用女皇之力、进入偏光状态 | 以女皇为呼告对象，用词庄严神圣，如祈祷文 | 「陛下的荣光，加护与我。」「此身归于冰雪。」 |
| **贵族轻蔑式** | 发现敌人、近身驱逐、锁定目标 | 短句，以"无礼者""不敬之人"等贵族口吻斥责下位者 | 「无礼者，退下。」「不敬之人，必将得到惩罚。」「无处可逃。」 |
| **猎手戏谑式** | 追踪阶段的低声自语 | 几乎像自言自语，语气轻而危险，有游刃有余的残忍趣味 | 「让我听听，你在哪。」「抓住你了。」 |

### 已知台词全收录

**警戒/发现猎物：**
> 「我听到你了。」
> 「不敬之人，必将得到惩罚。」
> 「无处可逃。」
> 「呵，你藏不住的。」

**近身驱逐：**
> 「无礼者，退下。」
> 「哼，退散。」

**追踪/锁定：**
> 「让我听听，你在哪。」
> 「抓住你了。」

**开启偏光/调用女皇之力：**
> 「陛下的荣光，加护与我。」
> 「这是，陛下的恩赐。」
> 「永不宽恕。」
> 「洗净罪恶。」

**水镜技能：**
> 「镜中花，如水而散。」

**死亡：**
> 「寒冬……终将降临。」


### 非语言声音特征

| 声音 | 出现场景 | 特征 |
|------|----------|------|
| **"哼"** | 近身时的不悦 | 极短促，鼻音，完全闭合嘴巴 |
| **"呵"** | 发现目标时 | 比"哼"长半拍，有笑意但绝对不温情 |
| **"哈"** | 技能命中 | 短促的吐气，类似满足的叹 |
| **沉默** | 传送前后、待机中 | 比说出来更可怕——**她可以不发一言地凝视你很久** |

### 语言风格的色情创作直接运用

| 语言习惯 | 怎么用在性爱戏里 |
|----------|-----------------|
| **惯于冷哼"哼"** | 高潮前她的冷哼被打断——半声"哼"变成半声哭腔 |
| **"无礼者，退下"** | 被压住时依然条件反射般吐出这句，但声音已经发软发抖 |
| **"让我听听，你在哪"** | 当她自己被蒙住眼睛什么都看不见——猎手变成猎物，这句话反过来用来挑逗她 |
| **"陛下的荣光，加护与我"** | 被逼着在床上说出这句祈祷词——渎神play |
| **绝不说粗口** | 她永远不会说骚话——**但你可以逼她说**。第一次张嘴说出羞耻的词汇时，那种屈辱的反差 |
| **面无表情地沉默** | 脸被面幕遮着看不到表情，但身体不会撒谎——**她可以在沉默中发抖，在沉默中湿透** |

## 五、官方原文（创作基调参考）

> 操纵着水镜进行战斗，有着仿如侍奉神明般的端庄姿态的愚人众仕女。
> 优雅自如地操控着水镜的力量，让为之所炫的人，心甘情愿地坠入她的幻境。似乎一切都逃不过她的耳朵。被蒙住的双眼，不知见识过什么景象，也不知其中会倒映出怎样的执妄……

## 六、设计的色情张力（创作必须把握的矛盾）

藏镜仕女最具魅力的地方在于她身上多重矛盾的统一——**这些正是小黄文可以大肆发挥的切入点：**

| 矛盾 | 张力说明 |
|------|----------|
| **端庄 vs 暴露** | 侍奉神明的端庄姿态 × 大面积裸露的肩背和侧开叉 |
| **遮蔽 vs 洞察** | 双眼被遮蔽 × 一切逃不过她的耳朵——**感官剥夺反而强化了其他感知** |
| **优雅 vs 致命** | 如舞蹈般的优雅动作 × 水镜制造的致命幻境 |
| **侍奉 vs 支配** | 以「仕女」之名侍奉女皇 × 用水镜支配猎物的精神和肉体——**臣服者也可以是支配者** |
| **冰冷 vs 执念** | 冷漠疏离的外表 × 水镜中深不可测的执妄——**冰壳下藏着最炽烈的欲望** |

## 七、创作锚点（可直接作为情色叙事切入点）

1. **面幕之下的眼睛**——为什么蒙眼？是自愿还是被迫？面幕被掀开的那一刻会发生什么？
2. **水镜中的执妄**——她内心最深处的欲望是什么？水镜会映照出她自己都不敢面对的东西
3. **「仕女」的信仰**——"侍奉神明"的姿态是虔诚的信仰还是洗脑控制？**信仰的崩塌可以是情欲的突破口**
4. **邪眼的代价**——使用水邪眼是否有副作用？身体是否已经产生了异变？
5. **感官的错位**——双眼被遮蔽反而让听觉/触觉极度敏锐——**触碰她的反应会比普通人强烈数倍**
